# cucumberjs-webdriverio

same as cucumberjs-webdriverjs just using webdriverio instead of webdriverjs

## how to run

Open terminal and run:
* $ ./node_modules/.bin/selenium-standalone start

Then open another terminal and run:
* $ ./run.sh
